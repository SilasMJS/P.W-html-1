HTML

O que é?

HTML(acrônimo para expressão inglesa HyperText Markup Linguage, que significa Linguagem de Marcação de Hipertexto) é Uma linguagem de marcação utilizada para prodzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

DreamWeaver

O que é?

O Macromedia DreamWeaver é um software de criação para a web criada pela Macromedia (agora Adobe Systems), que está atualmente na versão CS3. Versões iniciais da aplicação serviam como um simples editor HTML WYSIWYG ("What You See Is What You Get", ou "O que você vê é o que você tem"), porém versões mais recentes incorporaram notável suporte para muitas outras tecnologias web tais como XHTML, CSS, JavaScript e vários scripts de servidor. Desde o final dos anos 90, o DreamWeaver vem tendo um sucesso cresente e hoje domina cerca de 80% do mercado de editores HTML.
